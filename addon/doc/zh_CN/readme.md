# WinTeleport

本 NVDA 插件通过快捷键让你可以轻松地在 Windows 虚拟桌面之间移动当前焦点窗口。

## 功能特性
- 将窗口移动到左侧或右侧的虚拟桌面。
- 将窗口移动到指定编号的虚拟桌面。
- 支持在移动窗口后自动切换到目标桌面。
- 播报当前虚拟桌面的名称和编号。

## 默认快捷键
- `NVDA+Windows+D`: 播报当前虚拟桌面的信息（名称、编号及总数）。
- `NVDA+Control+Windows+左方向键`: 将焦点窗口移动到左侧桌面并切换过去。
- `NVDA+Control+Windows+右方向键`: 将焦点窗口移动到右侧桌面并切换过去。
- `NVDA+Control+Windows+1` 至 `9`: 将焦点窗口移动到第 1-9 个虚拟桌面。

*注意：如果你希望在移动窗口后保留在当前桌面（不跟随切换），可以在 NVDA 的“输入按键”对话框中的“虚拟桌面”类别下自行分配快捷键。*

## 数字快捷键限制
- 目前直接移动到指定编号的功能仅支持第 1 到第 9 个桌面，你只能分配包含 1 到 9 键的自定义按键组合。

## 已知问题与限制
- **兼容性**: 极个别采用非标准窗口框架的应用（例如传统版 QQ）可能无法通过该插件移动窗口。
- **焦点问题**: 当你将某个虚拟桌面上的最后一个窗口移动到其他桌面，且选择不跟随切换时，系统焦点可能会停留在“未知”对象上。

## 参考项目

本项目参考并集成了以下开源项目的实现逻辑：
- [pyvda](https://github.com/mirober/pyvda): 适用于 Windows 10/11 虚拟桌面管理的 Python 库。
- [MoveToDesktop](https://github.com/Eun/MoveToDesktop): 一个实用的窗口桌面移动工具。
